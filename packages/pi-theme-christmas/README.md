# 🎄 @codella/pi-theme-christmas 🎅

🎁 Festive Christmas theme for [Pi](https://pi.dev): evergreen panels, candy-cane red, ornament gold, frosty blue, snow white, and warm cocoa shadows. ✨

## 🎄 Install

```bash
pi install npm:@codella/pi-theme-christmas
```

## 🎅 Use

Select `christmas` in `/settings`, or add it to your Pi settings:

```json
{
  "theme": "christmas"
}
```

## 🎁 Package contents

```text
themes/christmas.json
```
