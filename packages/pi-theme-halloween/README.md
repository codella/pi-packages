# 🎃 @codella/pi-theme-halloween 👻

🦇 Spooky Halloween theme for [Pi](https://pi.dev): pumpkin orange, poison green, ghostly violet, blood red, moonlit text, and midnight haunted-house panels. 🕸️

## 🎃 Install

```bash
pi install npm:@codella/pi-theme-halloween
```

## 👻 Use

Select `halloween` in `/settings`, or add it to your Pi settings:

```json
{
  "theme": "halloween"
}
```

## 🕷️ Package contents

```text
themes/halloween.json
```
