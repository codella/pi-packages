# @codella/pi-theme-grayscale

Minimal grayscale theme for [Pi](https://pi.dev): monochrome panels, soft silver borders, bright text, and clear contrast without color hue.

## Install

```bash
pi install npm:@codella/pi-theme-grayscale
```

## Use

Select `grayscale` in `/settings`, or add it to your Pi settings:

```json
{
  "theme": "grayscale"
}
```

## Package contents

```text
themes/grayscale.json
```
