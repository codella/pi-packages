# @codella/pi-theme-cyberpunk

Neon cyberpunk theme for [Pi](https://pi.dev): cyan, hot pink, acid green, electric blue, and yellow on a dark terminal background.

## Install

```bash
pi install npm:@codella/pi-theme-cyberpunk
```

## Use

Select `cyberpunk` in `/settings`, or add it to your Pi settings:

```json
{
  "theme": "cyberpunk"
}
```

## Package contents

```text
themes/cyberpunk.json
```
